Primitive Types in R
================

``` r
l <- FALSE
n <- 12.333
i <- 64L
cx <- 3+2i
cc <- 'programming'

class(l) 
```

    ## [1] "logical"

``` r
class(n)
```

    ## [1] "numeric"

``` r
class(i)
```

    ## [1] "integer"

``` r
class(cx)
```

    ## [1] "complex"

``` r
class(cc)
```

    ## [1] "character"
